# CeramIQ

App ligera de consulta experta para ceramica dental by Miguel Arroyo.

Incluye captura guiada, subida explicita de guia de color, tarjeta BigColor Cera, foto polarizada, evidencia comparativa objetivo vs prueba, CIELAB por tercios, diagnostico optico, Delta E, criterio de material, estrategia optica, receta con cuatro masas por tercio cuando proceda y biblioteca CeramIQ.

Estado: app demostrable para curso. No declara medicion clinica absoluta sin calibracion real.
