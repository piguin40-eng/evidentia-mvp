# Protocolo Miguel - bloqueo de sustrato oscuro en carilla feldespatica

Fuente: aprendizaje clinico-tecnico aportado por Miguel, 2026-05-29.
Estado: criterio experto de laboratorio, no paper externo.

## Indicacion

Carilla en anterior con sustrato de tallado muy oscuro, incluso casi negro, cuando se necesita copiar un diente vecino bleach, muy luminoso y translucido.

## Criterio principal

En estos casos, Miguel considera preferible resolver la restauracion como feldespatica pura, no intentando compensar el sustrato negro solo con una estratificacion translucida convencional.

La clave es crear primero una capa ultrafina de bloqueo optico, aproximadamente 0.1 mm, y despues estratificar el color/luminosidad sobre una base ya neutralizada.

## Primera coccion de bloqueo

Realizar una coccion tipo wash muy fina con:

- Deep Dentin bleach.
- Opaquer de metal bleach.
- Una pequena cantidad de pasta glaze mezclada.

Objetivo de la capa:

- Bloquear el sustrato negro.
- Subir el valor inicial.
- Evitar que la translucidez posterior agrise la restauracion.
- Crear una base luminosa y controlada para poder copiar despues el bleach del diente vecino.

Espesor orientativo:

- Aproximadamente 0.1 mm.

## Estrategia posterior

Una vez bloqueado el sustrato oscuro con esa primera capa, el caso se vuelve mucho mas manejable:

- El color bleach puede construirse con masas feldespaticas de alto valor.
- La translucidez se puede trabajar de forma selectiva sin que el fondo negro contamine.
- El incisal puede recuperar vida, halo y profundidad sin perder luminosidad general.

## Regla para Yolito

Cuando Miguel plantee un caso de carilla sobre sustrato muy oscuro/casi negro y objetivo bleach translucido, Yolito debe priorizar esta estrategia:

1. Feldespatica pura si el caso y el espacio lo permiten.
2. Wash inicial ultrafino de bloqueo.
3. Mezcla de Deep Dentin bleach + opaquer de metal bleach + pequena cantidad de pasta glaze.
4. Capa aproximada de 0.1 mm para neutralizar el negro.
5. A partir de esa base, estratificar el color objetivo por tercios.

Yolito debe etiquetar este protocolo como aprendizaje clinico de Miguel y no presentarlo como recomendacion IFU universal.
