## Microdemo De 5 Minutos

Usar si el candidato tiene poco tiempo o todavia no hay confianza para una demo completa.

1. Decir limite en 15 segundos: memoria contextual con fuentes, no diagnostico ni decision clinica.
2. Mostrar estado local: datos en el nodo, IA externa desactivada o bajo control del cliente.
3. Hacer una pregunta sobre perdida de conocimiento: protocolo, aprendizaje, fuente o criterio.
4. Mostrar primero las fuentes y despues el resumen.
5. Preguntar: "Que 3 fuentes tuyas harian que esto fuera obviamente util en 30 dias?"
6. Cerrar solo con siguiente paso verificable: completar candidato, datos permitidos, fecha y permiso de Miguel.

## Guion De 10 Minutos

1. Abrir Evidentia Local Node y decir el limite: memoria contextual, no diagnostico.
2. Mostrar que los datos estan en el nodo local y que la IA externa esta desactivada por defecto.
3. Hacer una pregunta de demo sobre conocimiento acumulado.
4. Mostrar fuentes antes de hablar de valor.
5. Exportar o mostrar el knowledge bundle.
6. Preguntar que 3 fuentes reales deberia recuperar para que el piloto sea util.
7. Cerrar con piloto fundador solo si entiende limites, fuentes y datos autorizados.

## Preguntas De Calificacion

1. Que conocimiento pierdes hoy entre WhatsApp, fotos, PDF, notas, audios o carpetas?
2. Que decision, protocolo o aprendizaje tarda demasiado en reconstruirse?
3. Que 3 fuentes deberia recuperar Evidentia para que el piloto sea obviamente util?
4. Quien valida si una respuesta con fuentes es suficiente?
5. Que datos no deben entrar nunca en el piloto?
6. Tienes API/proveedor de IA propio o prefieres empezar en modo local con fuentes?

## Criterios Para Decir Si

- Tiene dolor real de memoria, trazabilidad o recuperacion de conocimiento.
- Acepta datos ficticios, anonimizados o autorizados.
- Entiende que la decision sigue siendo humana.
- Puede dedicar 30 dias y una revision semanal breve.
- Acepta el precio fundador: 500 EUR setup + 50 EUR/mes soporte basico.

## Criterios Para Decir No

- Quiere diagnostico, recomendacion clinica automatica o promesa medica.
- No puede autorizar/minimizar datos.
- Quiere que Miguel/Helix pague tokens externos o custodie datos sensibles.
- No hay responsable claro para validar fuentes.
- No acepta que el piloto se pare si no recupera fuentes utiles.
